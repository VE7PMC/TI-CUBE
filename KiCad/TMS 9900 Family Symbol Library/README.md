# Description

This folder contains a KiCad symbol library for the following [Texas Instruments TMS 9900 family](https://archive.org/details/bitsavers_tiTMS99001emsDesignandDataBook1ed_39738539) parts:

Component   | Description
---------   | -----------
TMS 9900    | NMOS 16-Bit Microprocessor
TMS 9901    | Programmable System Interface
TMS 9902    | Asynchronous Communications Controller
TMS 9903    | Synchronous Communications Controller
TIM 9904(A) | 4-Phase Clock Driver
